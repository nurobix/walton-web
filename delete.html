Suppression de vos données — Walton

Application Walton — éditée par Nurobix

Données stockées par Walton

Walton stocke les données suivantes uniquement sur votre appareil, dans une base de données locale :

  • Vos spots de pêche personnels (nom, position GPS, notes, photos)
  • Votre carnet de touches (espèce, poids, taille, photo, conditions météo)
  • Vos stations Vigicrues favorites
  • Vos préférences d'utilisation

Aucune de ces données n'est transmise à un serveur externe. Elles sont stockées exclusivement sur votre téléphone.

Comment supprimer vos données

Option 1 — Depuis l'application (recommandé) :

  1. Ouvrez Walton
  2. Allez dans Paramètres
  3. Appuyez sur "Supprimer toutes mes données"
  4. Confirmez la suppression

Toutes vos données sont effacées immédiatement et définitivement.

Option 2 — Désinstallation de l'application :

La désinstallation de Walton supprime automatiquement toutes les données locales de votre appareil.

Option 3 — Demande par email :

Pour toute question concernant vos données, contactez-nous à contact@nurobix.io. Nous vous répondrons sous 72 heures.

Durée de conservation

Walton ne conserve aucune donnée sur ses serveurs. Toutes les données sont supprimées immédiatement lors de la suppression dans l'appli ou de la désinstallation. Aucune durée de conservation supplémentaire ne s'applique.

Phase 2 — Compte utilisateur

Lorsque Walton proposera la création d'un compte (synchronisation cloud), cette page sera mise à jour avec la procédure de suppression du compte et des données associées.
